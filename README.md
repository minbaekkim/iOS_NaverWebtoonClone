## 1. 프로젝트 소개
```
AutoLayout 기능을 적응하기 위해 만든 화면 구성 클론
```

## 2. 사용한 기술

- Scroll View 적용
- Horizontal Stack View와 Vertical Stack View를 조합하여 테이블 생성


## 3. 설치 방법

1. 다운로드 후 압축을 푼다
2. .xcodeproj 확장자 파일을 실행한다
